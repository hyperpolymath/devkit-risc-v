== Language Policy

This project follows the **Hyperpolymath Standard** (see `.claude/CLAUDE.md`).

.Permitted Stack
[cols="1,2",options="header"]
|===
| Technology | Use Case

| Rust | Systems, CLI, WASM, performance-critical code
| ReScript | Application code (compiles to JS)
| Deno | Runtime & package management
| Gleam | Backend services (BEAM/JS target)
| Tauri 2.0+ / Dioxus | Mobile apps (no Kotlin/Swift)
| Guix / Nix | Package management
|===

.Explicitly Banned
* TypeScript, Node.js, npm, Bun, Go, Java, Kotlin, Swift, Flutter
